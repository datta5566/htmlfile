# DK AI Short Video Studio

यह GitHub tool Instagram Reels, YouTube Shorts, WhatsApp Status और LinkedIn short videos के लिए VidRush-ready prompt तैयार करता है।

## कैसे इस्तेमाल करें

1. `index.html` खोलें।
2. Topic, platform, duration, language, style, audience और main points भरें।
3. **Generate VidRush Prompt** दबाएँ।
4. Prompt copy करें।
5. **Open VidRush** दबाकर VidRush में prompt paste करें।
6. Generation से पहले VidRush के Quote Statement में model, duration, language और credits check करें।

## Features

- 15, 30, 45 और 60 second short-video briefs
- Instagram, YouTube, WhatsApp और LinkedIn formats
- Hindi, Hindi-English, English और Marathi
- Professional, viral, cinematic, educational, motivational और promotional styles
- Strong hook, captions, B-roll, music, CTA और safety instructions automatic
- Browser में preset save
- Last prompt local storage में save
- VidRush और official quick-start guide के direct buttons

## Integration सीमा

यह launcher/prompt integration है। VidRush की public video-generation API उपलब्ध न होने तक GitHub page सीधे video render नहीं कर सकता। User VidRush account में login करके generated prompt से video बनाता है।
